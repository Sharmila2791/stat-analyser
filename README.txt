# TDS Challan Extractor
TDS Challan Extractor is a simple web-based tool that allows users to upload one or more TDS challan PDF files and extract key details like Nature of Payment, Amount (in Rs.), Date of Deposit, and Challan Number. The extracted data is displayed in a table and can be exported to Excel for record-keeping or further analysis.
## Features

- Upload and process multiple TDS challan PDFs at once
- Automatically extract the following information:
  - Filename
  - Nature of Payment
  - Amount (in Rs.)
  - Date of Deposit
  - Challan Number
- Display extracted data in a responsive table
- Export the table data to an Excel file
- 100% client-side; no server or backend required
